# Security Policy

## Scope

Security reports are welcome for the live service at
[kurrentschrift.ink](https://kurrentschrift.ink) and for the code in this
repository.

## Reporting a vulnerability

Please report vulnerabilities privately to **admin@kurrentschrift.ink**.
Do not open public GitHub issues for security problems.

You will receive an acknowledgement within a week. This is a small
open-source portfolio project, so fixes are handled on a best-effort basis.
